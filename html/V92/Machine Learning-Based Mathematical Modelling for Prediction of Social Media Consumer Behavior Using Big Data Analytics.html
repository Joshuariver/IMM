Machine Learning-Based Mathematical Modelling for Prediction of Social Media Consumer Behavior Using Big Data Analytics

Chaudhary · Alam · Al-Rakhami · Gumaei | Journal of Big Data (2021) 8:73

Decision Tree (98.29%) Big Data Analytics Social Media n=5279 records 7 ML Models

📄 STEP 1 — Abstract 번역

ENGLISH

Social media is popular in our society right now. People are using social media platforms to purchase various products. We collected the data from various social media platforms (Facebook, Twitter, LinkedIn, YouTube, Instagram, Pinterest). We developed mathematical modeling using machine learning to predict consumer behavior on the social media platform. 80% of data are used for training purposes and 20% for testing.

한국어 번역

소셜 미디어(Facebook, Twitter, LinkedIn, YouTube, Instagram, Pinterest)에서 소비자 행동 데이터를 수집하였다. 이상값·잡음·오류·중복 레코드를 제거하는 전처리 후, 머신러닝 기반 수학적 모델을 개발하여 소비자 행동을 예측하였다. 80% 훈련/20% 테스트 분할 결과 최고 정확도 98%를 달성하였다.

⚠️ STEP 5 — Limitation 분리

A. 논문이 스스로 인정한 한계

① 일별 데이터 적용 불가
"The limitation of this model is that it will not work on daily basis consumer data. If this model is used on daily basis data, the result will be very poor." (Conclusion, p.18-19)
② 분산 처리 미구현
클라우드 분산 처리(Hadoop MapReduce) 활용 권고만 있고 본 연구에서는 미구현 (Conclusion, p.19)
B. 논문이 언급하지 않은 비판적 의심점 (펼치기)
극소 표본 문제Instagram 11개, Pinterest 15개 레코드 — 통계적 대표성 극히 불충분. 98%+ 정확도는 과적합(overfitting) 강력 의심
종속변수 혼합Likes/Followers/Visits/Downloads를 단일 수치로 통합 → 이질적 지표 동일 척도 취급의 개념적 오류
B2B 관련성 부재B2C 소비자 데이터 기반. B2B 맥락(기업 구매자, DMU)에 적용 시 유효성 미검증
시계열 특성 무시소셜 미디어 데이터는 시간적 패턴을 가지나 무작위 80/20 분할로 데이터 누수(data leakage) 가능
수집 방법 불투명데이터 수집 시점, 기간, 크롤링/API 방식 미명시 → 재현 어려움
DTR 해석 부재98.29% 정확도 주장이나 feature importance·교차검증 결과 미제시 → 과적합 여부 판단 불가
